# Personal Project Showcase App

A modern, responsive web application for showcasing personal projects. Built with React, this app allows users to add, view, search, and delete projects.

## Features

- ✅ **Add Projects** - Submit project title and description
- ✅ **View Projects** - Display projects in a responsive grid layout
- ✅ **Search Projects** - Filter projects by title or description
- ✅ **Delete Projects** - Remove projects with confirmation dialog
- ✅ **Form Validation** - Required fields and character limits
- ✅ **Responsive Design** - Works on mobile, tablet, and desktop
- ✅ **Unit Tests** - Comprehensive testing with Jest & React Testing Library
- ✅ **Accessible** - ARIA labels and keyboard navigation support

## Technology Stack

- **React 18** - UI library with Hooks
- **Vite** - Next-generation build tool
- **CSS Modules** - Component-scoped styling
- **Jest & React Testing Library** - Testing framework
- **ESLint** - Code linting
